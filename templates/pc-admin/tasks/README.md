# Tasks

Root task files are stage-gated.

- Copy `tasks/backlog.template.md` to `tasks/backlog.md` after `requirements-confirmed`.
- Copy `tasks/sprint-01.template.md` to `tasks/sprint-01.md` at `implementation-ready`.

Frontend and backend feature-level task files live under each side's `SPECS/FEATURES/<feature-slug>/tasks.md`.

Do not rename or edit `*.template.md` files as active task plans. Keep them reusable for future stages and generated projects.
