# 系统配置模块需求

## 1. 模块目标

提供基础参数配置能力，用于维护系统名称、登录页标题、文件上传限制、Token 有效期等可配置项。

## 2. 查询条件

- 配置编码
- 配置名称
- 配置状态

## 3. 列表字段

- 配置编码
- 配置名称
- 配置值
- 配置状态
- 配置说明
- 更新时间
- 操作

## 4. 维护字段

- 配置编码
- 配置名称
- 配置值
- 配置状态
- 配置说明

## 5. 业务规则

- 配置编码必须唯一
- 配置项支持启用和停用
- 配置变更需要记录操作日志
- 第一阶段采用数据库配置
- 第一阶段不强制实现复杂的动态热更新
- 对运行态敏感的配置变更可在下次服务启动或下次读取时生效

## 6. 预置配置

- 系统名称
- 登录页标题
- 文件上传大小限制
- 允许上传文件类型
- Access Token 有效期
- Refresh Token 有效期

## 7. Demo 要求

- 支持配置项查询、编辑、启用和停用
- 修改系统名称后，首页顶部栏或登录页标题可体现变化
- 修改文件上传限制后，附件上传校验应按新配置执行
